# Chicago Crime Data Analysis

## Project Overview

This project is an exploratory data analysis project using Chicago crime data.

The goal was to practice working with a real-world public dataset and use Python to find patterns in crime types, trends, and locations. I wanted this project to help me improve my data cleaning, pandas, visualization, and analysis skills.

## Business Problem

Large public safety datasets can be difficult to understand without cleaning and analysis.

The main question for this project was:

**What patterns can be found in Chicago crime data using exploratory data analysis?**

This type of analysis can help:

- Understand common crime categories
- Identify trends over time
- Explore location-based patterns
- Support public safety discussions with data
- Practice real-world data analysis skills

## Dataset

This project uses Chicago crime data.

The dataset includes information such as crime type, date, location, arrest status, and other public safety details.

Note: The dataset file is not included in this repository due to file size considerations.

## Tools and Technologies

- Python
- pandas

- Jupyter Notebook
- Data cleaning
- Exploratory Data Analysis

## Project Workflow

1. Loaded and inspected the dataset
2. Checked columns, data types, and missing values
3. Cleaned and prepared the data
4. Explored crime categories
5. Analyzed crime trends
6. Created visualizations using matplotlib
7. Summarized key findings from the data

## What I Learned

This project helped me understand how to work with a large real-world dataset.

I practiced using pandas for data cleaning and analysis, matplotlib for visualization, and exploratory data analysis to find useful patterns from raw data.
